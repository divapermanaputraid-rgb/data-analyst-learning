## Data Analyst Learning Repository
### Purpose

This repository documents a structured learning process toward a data analyst role. It emphasizes ***reasoning, metric thinking, and business impact***, not just tool usage.

### How to Read This Repo

* Folders represent ***skill domains*** (SQL, Excel, Business Thinking, etc.)

* Files represent ***living logs*** that evolve over time

* Portfolio-ready projects are separated from learning notes

### Repository Structure
```text
00-foundation/        # Learning rules, tools, mindset
01-sql/               # SQL fundamentals, notes, queries
02-excel/             # Excel analysis & reporting (single evolving log)
03-python/            # Python (later phase)
04-statistics/        # Statistics & experimentation basics
05-business-thinking/ # Metrics, monetization, decision framing
notes/                # Daily log & mistakes
portfolio-projects/   # Polished, end-to-end projects
roadmap.md            # Learning roadmap & milestones
```
